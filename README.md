# aaronmeza/homebrew-tap

Personal Homebrew tap for tools published under [github.com/aaronmeza](https://github.com/aaronmeza).

## Use

```bash
brew tap aaronmeza/tap
brew install <formula>
```

Or in a single command:

```bash
brew install aaronmeza/tap/<formula>
```

## Available formulae

| Formula | Description |
|---|---|
| [`claude-tasks`](Formula/claude-tasks.rb) | Per-project persistent task tracking for Claude Code with cross-project prioritisation. [Source](https://github.com/aaronmeza/claude-tasks) |
